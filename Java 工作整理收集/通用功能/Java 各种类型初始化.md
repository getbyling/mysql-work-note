Java 各种类型初始化
===================

>Set 变量初始化
```java
import java.util.Set;

public class test {
    public static void main(String[] args) {
        Set set = new HashSet();
    }
}
```

>List 变量初始化
```java
import java.util.List;

public class test {
  public static void main(String[] args) {
      List list = new ArrayList<>();
  }
}
```

>Map 变量初始化
```java
import java.util.Map;

public class test {
  public static void main(String[] args) {
      Map<String, Object> params = new HashMap<>();
  }
}
```
