mvn install 使用
================
>mvn install 简单安装：
```text
mvn install
```

>mvn package 打包
```text
l@l-PC MINGW64 /d/GaiayCode/origincode/extends (develop)
mvn clean package -Dmaven.test.skip=true
```

>mvn install 安装
```text
l@l-PC MINGW64 /d/GaiayCode/origincode/zm/zm-sync (develop)
mvn clean install -Dmaven.test.skip=true
```
